DISABLING A CORE
══════════════════════════════════════════════════════════════════════════════

  To disable a core (here, cpu0), simply run:

  sudo echo 0 >/sys/devices/system/cpu/cpu0/online

  See:  https://unix.stackexchange.com/questions/145645/disabling-cpu-cores-on-quad-core-processor-on-linux
